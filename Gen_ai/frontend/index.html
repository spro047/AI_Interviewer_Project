About this engine

How answers are scored

A weighted hybrid of three independent signals:

  • TF-IDF · 30% Keyword overlap with the ideal answer.
  • BERT · 50% Semantic similarity via all-MiniLM-L6-v2 embeddings.
  • LLM · 20% Concept coverage, depth, clarity, missing concepts.

Final = 0.3·TF-IDF + 0.5·semantic + 0.2·LLM

Difficulty adapts after every answer: > 7.5 → harder, < 4 → easier.

Models in use

  • LLM — Qwen/Qwen2.5-7B-Instruct via HuggingFace Inference API
  • Embeddings — sentence-transformers/all-MiniLM-L6-v2 (local)
  • Adaptive — rule-based, smoothed 0.7·current + 0.3·prev
